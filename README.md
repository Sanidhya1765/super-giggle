# 🎮 CLI Quiz Game

A fun and interactive Command Line Quiz Game built with **JavaScript (Node.js)**.

Test your programming knowledge directly from the terminal and see your final score!

---

---

## ✨ Features

- 🎯 Multiple-choice quiz
- 🖥️ Beautiful CLI interface
- ✅ Instant answer checking
- 🏆 Final score display
- ⚡ Fast and lightweight
- 💻 Built with JavaScript (Node.js)

---

## 📦 Installation

Clone this repository

```bash
https://github.com/Sanidhya1765/super-giggle
```

Go to the project folder

```bash
cd flaks
```

Install dependencies

```bash
npm install
```

---

## 🚀 Run the Game

```bash
node index.js
```

or

```bash
node .
```
